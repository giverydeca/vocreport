解決ステータスと未完了分析

解決は5,656件（56.6%）にとどまり、折り返し・調査中が3,987件（39.9%）と高水準。大区分別では「クレーム（未完了率87.8%）」「担当者不在（91.4%）」が特に対応困難な領域となっている。

解決ステータス分布
大区分別 未完了率（折り返し＋希望通らず）
  • クレーム（未完了率87.8%） — 破損369件・パーツ不足262件が主体。現品調査が必要で電話では解決不可。受付フローの標準化と証跡取得が急務
  • 担当者不在（91.4%） — 折り返し対応で完結しているが、初回解決率の向上が求められる
  • 納期・注文・商品仕様（40〜44%） — 件数が多い区分で未完了が4割超。データ反映遅延や在庫情報不足が主因
  • カタログ・DM（未完了率7%） — 登録情報の変更・停止処理で解決しやすい。参考にできる対応モデル
顧客の声 — 解決しにくいケース
クレーム / パーツ不足（折り返し・調査中）
「本日木製のフラワーボックスが届いたんですけど、6個頼んでるんですけど5個しか届いてなくて。メール見ても6個注文になってるはずなんですけど。」
→ 注文データと出荷数の不一致。現品確認が必要で折り返し対応が発生。受付時の証跡（写真・伝票番号）取得フローの整備が有効。
ログイン / セキュアキーエラー（解決不可）
「ログインしようと思うと、セキュアキーに誤りがあります。こちらからログアウトをお願いしますという画面に出てきて、ログアウトもできないような状況なんですけれども。」
→ システム起因でオペレーターも対処不可。エラー画面の改善とサポートへの誘導が必要。
クレーム / 同梱漏れ（折り返し・調査中）
「今日商品届いたんですが、バーセットが3本しか入っていなくて、スプリングポール本体とスリムフックハンガーが同梱されていませんでした。」
→ 組み立て品の同梱漏れ。検品プロセス見直しと受付後の対応基準明確化が必要。
